# OpenMRP — Frontend

The public website for [openmrp.in](https://openmrp.in): barcode search, the
`/p/{barcode}` product pages (SEO-indexed), brand/category browse, the multilingual
view, and the data/license page.

**Status:** 🚧 Not built yet.

**Planned:** Next.js / Astro on Cloudflare Pages, reading the public API from
[`../backend`](../backend). Build order is **backend → seed data → frontend → dump**,
so this comes online once the backend has real data to render.
