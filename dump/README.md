# OpenMRP — Open Data Dumps

This folder will hold the **ODbL-licensed open data exports** (JSONL / CSV) of the
OpenMRP product database, plus the export tooling that generates them.

**Status:** 🚧 Not built yet.

The dump is produced from the database once it has data — see the build order
(**backend → seed → frontend → dump**). Publishing these dumps openly is also how we
satisfy the ODbL share-alike obligation by design.

All data here is published under the [Open Database License (ODbL 1.0)](../LICENSE-DATA).
